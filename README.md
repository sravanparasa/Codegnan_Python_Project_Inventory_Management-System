# Codegnan_Python_Project_Inventory_Management-System

Overview
------------------------------------------------------------------------------------------------------------------------------------------------------------------
The Inventory Management System is a menu-driven Python application developed to manage product inventory efficiently. It allows users to add, view, search, update, and delete products, manage stock purchases and sales, and generate inventory reports. The project is built using Python modules and functions without using databases or file handling.

Features
-------------------------------------------------------------------------------------------------------------------------------------------------------------------
User Login Authentication
Add New Product
View All Products
Search Product by ID
Update Product Details
Delete Product
Purchase Stock
Sell Product
Inventory Reports
Exception Handling for Invalid Inputs

Technologies Used
-------------------------------------------------------------------------------------------------------------------------------------------------------------------
Python
Functions
Modules
Lists & Nested Lists
Loops
Conditional Statements
Exception Handling
